## 2算法
算法是对特定问题求解步骤的一种描述，是指令的有限序列。
算法特性：
* 有穷性
* 确定性
* 可行性
* 有输入
* 有输出

算法设计目标：
* 正确性
* 健壮性
* 高效性
* 可读性

算法描述方法：
1. 自然语言
2. 流程图
3. 程序设计语言
4. 伪代码
### 2.1 算法分析
#### 2.1.1 时间代价

#### 2.1.2 空间代价
算法执行时占用的存储空间量：空间复杂度O(n)
